# 📚 Book Management System

A simple and interactive **command-line Book Management System** built with **Python**.  
This project helps users manage books by performing basic CRUD operations (Create, Read, Update, Delete).

---

## ✨ Features

✅ Add a new book  
✅ View all books  
✅ Search for a book  
✅ Update book details  
✅ Mark a book as read  
✅ Delete a book  
✅ User-friendly menu-driven interface
